# Project Rules & Preferences

## Tech Stack
- Frontend: HTML5, Tailwind CSS (via CDN)
- Editor Origin: Started in Cursor

## Coding Standards
- **Naming:** Use camelCase for JavaScript and kebab-case for CSS classes.
- **Style:** Keep things modular. If a section gets too long, suggest a way to break it up.
- **Design:** Lean towards "Modern Minimalist"—lots of whitespace and Inter font.

## Workflow
- Always check the `index.html` file before suggesting changes to CSS.
- Provide a preview using the Artifacts tool when changing layouts.
